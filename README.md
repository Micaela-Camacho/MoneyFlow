MoneyFlow 💸

MoneyFlow es una aplicación móvil diseñada para facilitar la gestión de finanzas personales. 
Permite a los usuarios llevar un control detallado de sus ingresos, gastos y ahorros de manera intuitiva y segura.

🚀 Características principales

Gestión de Finanzas: Registro de gastos fijos, variables e ingresos extra.

Arquitectura Robusta: Desarrollada bajo el patrón MVVM (Model-View-ViewModel) para asegurar un código escalable y fácil de mantener.

Persistencia Local: Uso de Room Database para almacenar la información de forma segura y eficiente en el dispositivo.

Interfaz Moderna: Interfaz de usuario construida completamente con Jetpack Compose, ofreciendo una experiencia fluida y nativa.

Privacidad: Aislamiento de datos real por usuario.

🛠 Stack Tecnológico

Lenguaje: Kotlin.

UI: Jetpack Compose.

Arquitectura: MVVM (Model-View-ViewModel).

Base de Datos: Room (Persistencia local).

Asincronía: Coroutines y Flow.

📱 Instalación

Descarga el archivo .apk desde la sección de [Releases] de este repositorio.

Transfiere el archivo a tu dispositivo Android.

Habilita la instalación de "Orígenes desconocidos" en la configuración de seguridad de tu teléfono.

Instala y ¡listo!

⚙️ Configuración para Desarrolladores

Si deseas compilar el proyecto desde Android Studio:

Asegúrate de tener instalado JDK 17 o superior.

Clona este repositorio.

Abrí el proyecto en Android Studio.

Sincroniza las dependencias con Gradle y realiza un Rebuild Project.
